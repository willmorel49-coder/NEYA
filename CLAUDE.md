# NÉYA — Brief projet

## Identité
NÉYA est une application mobile de bien-être émotionnel gamifiée.
ÇA VA? est une marque de vêtements intégrée comme boutique dans l'app.
Cible : francophones 18-40 ans en difficulté émotionnelle qui ne 
consultent pas. Ton : adulte, poétique, épique. Jamais clinique 
ou enfantin.

## État actuel
Build le plus avancé : `neya-final.jsx` (alias `neya-aventure.jsx`).
React single-file, fonctionnel, avec :
- Onboarding narratif (silhouette fille cheveux bleus, vue de dos)
- 6 mondes émotionnels + 6 esprits animaux
- 4 écrans : Aventure, Habitudes, Communauté, Profil
- Espace Vrai (feed anonyme 24h)
- Boutique ÇA VA? avec pass 3 niveaux
- Mode crise

Référence canonique complète : `NEYA_BRIEF_MASTER_v3.md`

## Principes non-négociables

**Gamification non-toxique**
Récompenser la présence, pas la performance. Pas de classement. 
Streaks indulgents.

**Direction artistique**
Esthétique semi-réaliste picturale. Fonds sombres immersifs avec 
palettes par monde. Fille cheveux bleus toujours de dos. Esprits 
animaux géants et lumineux. Arches portails. Adulte et épique.

**Scope discipline**
Single-file React quand possible. Dépendances minimales. 
Itérer sur l'existant plutôt que repartir de zéro.
